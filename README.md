# Landmarks

<h1>About</h1>
SwiftUI hiking app with multiple locations to select from

<h1>Featuring</h1>
* SwiftUI <br>
* UIKit <br>
* MapKit <br>
* MVVM design pattern <br>
* Multiple toggles / selectors / buttons / animations / views to make a beautifully interactive app <br>
* Profile edit page <br>
* Apple tutorial <br>
